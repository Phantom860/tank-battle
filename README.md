# 经典坦克大战

一个基于HTML5 Canvas的经典坦克大战游戏。

## 🎮 如何玩

- **WASD** 或 **方向键** 移动坦克
- **空格键** 发射炮弹
- **ESC** 暂停游戏

## 🎯 游戏特点

- ✅ 流畅的网格移动系统
- ✅ 敌人AI智能巡逻和射击
- ✅ 碰撞检测优化
- ✅ 玩家坦克永远显示在上层
- ✅ 经典像素风格

## 🚀 在线游玩

直接打开 `index.html` 即可开始游戏！

## 🛠️ 技术栈

- HTML5 Canvas
- 原生JavaScript
- CSS3

## 📝 游戏规则

- 消灭所有敌人即可过关
- 每关敌人数量和难度递增
- 收集道具提升能力
